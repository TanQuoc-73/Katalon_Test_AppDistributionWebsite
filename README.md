App Distribution Website – Test Automation Portfolio

Automation testing project for a web-based application distribution platform similar to Steam or App Store.

This repository demonstrates automated UI testing practices using Katalon Studio, focusing on maintainable test architecture, reusable components, and structured test scenarios for validating critical user flows.

Project Purpose

The objective of this project is to design and implement an automated testing framework that validates the functionality and stability of an application distribution website.

The project focuses on:

End-to-end user journey validation
Automation of repetitive UI test cases
Maintainable test architecture
Scalable test suite design

This project is intended as a QA Automation portfolio project demonstrating practical testing skills.
